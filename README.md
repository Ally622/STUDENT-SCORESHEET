# Student Score Sheet

This project is a simple Java console program that collects student marks and produces a complete performance report. The user enters the school details and the marks for each student, and the program calculates totals, averages, and grades using arrays and loops.

The main purpose of the project is to practice basic Java concepts such as user input, arrays, loops, and conditional statements.

---

## **What the Program Does**

### **1. Collects School Information**
- School Name  
- Teacher Name  
- Grade  
- Year  

### **2. Captures Student Marks**
For each student, the program records:
- Name  
- English  
- Math  
- History  
- Geography  
- Science  
- Programming  

### **3. Calculates for Each Student**
- Total marks  
- Average  
- Letter grade (A, B, C, D, F)

### **4. Displays a Full Scoresheet**
- Student marks  
- Totals and grades  
- Class averages  
- Number of students per grade
